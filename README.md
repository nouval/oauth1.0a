## Usage

###Work with https (Node.js)

Depencies
```js
var oauth = require('serve-oauth');
var https = require('https');
```

Init
```js
var oauth_opts = {
		consumer: {
			public: '*****',
			secret: '*****'
		},
		signature_method: 'hmac-sha256',
	};
	var request_data = {
		url: 'https://apiu3.servevirtual.net/v1/utilities/authtest',
		method: 'GET'
	};
```
